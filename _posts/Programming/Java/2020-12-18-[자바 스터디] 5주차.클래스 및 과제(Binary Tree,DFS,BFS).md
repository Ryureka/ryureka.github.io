- ### int value, Node left, right를 가지고 있어야 합니다.

- {:.nostyle} ### 1.1
~~~ java
class Node {
    int value;
    Node left;
    Node right;
    public Node(int value, Node left, Node right){
        this.value = value;
        this.left = left;
        this.right = right;
    }
}
~~~
